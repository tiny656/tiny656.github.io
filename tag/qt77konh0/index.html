Young氏矩阵

2014-07-09 Data Structure

定义

一个m\*n的Young氏矩阵是指，每一行数据都是从左到右排好序，每一列的数据也都是从上到下排好序。其中也可能存在一些INF的数据，他们表示不存在的元素，一个m×nm \times nm×n的Young氏矩阵最多用来存放 r <= m\*n个元素。

并查集(Union Find Sets)

2014-07-08 Data Structure

并查集是什么？

并查集是一种树型的数据结构，用于处理一些不相交集合（Disjoint Sets）的合并及查询问题。常常在使用中以森林来表示。
集就是让每个元素构成一个单元素的集合，也就是按一定顺序将属于同一组的元素所在的集合合并。
